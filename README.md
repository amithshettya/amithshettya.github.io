# Personal Website

## Project info

I am learning a lot of technologies and would like the document and share what I have learnt. Since I use a lot of text editor, I thought it would be nice to have my personal website to be in text editor theme. A perfect way to display that I am a software engineer, isn't it? Additionally, I can organize my blog in folder, It's easier to follow all the blogs if it's organized. Yes, I am a sucker for organization.

Follow these steps:

npm is required for this project. So, my recommendation is to install nvm and they install the latest version  of npm.
```sh
npm i

npm run dev
```


## Web tech used in this project

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
